*Fusion Editor TKinter (Fusion Programming Language) - version:  Beta 1.0

Custom your Syntax Highlighting at: "dictionary.py"
You can also explore the Fusion Editor (TKinter) source code at: "fusion_editor.py"


Requires to work:
    - Python 3.x
    - Tkinter


--- LINUX ---
1. Install on Ubuntu (Debian):
    sudo apt instal python3-tk -y

2. Install on Fedora (Red Hat):
    sudo dnf install python3-tkinter

3. Install on CentOS (Red Hat):
    sudo yum install -y tkinter tk-devel

4. Install on Arch Linux (Manjaro):
    sudo pacman -S tk
